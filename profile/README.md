## 🤔 고봉밥은 어떤 팀인가요?
- 저희는 경기대학교 `오픈소스SW 실습` 수업의 프로젝트를 통해 만났어요.
- `팀원들의 넘치는 열정을 한 곳에 모두 담기가 힘들다` 라는 의미에서 **고봉밥**이라고 팀 이름을 지었어요.
- 주로 교내의 불편한 사항이나 시스템을 개선하기 위해 웹 서비스를 제작하고 있어요.
